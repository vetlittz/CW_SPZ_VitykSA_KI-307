# Windows Backup Utility  

Цей репозиторій містить програму для автоматичного резервного копіювання файлів та даних у Windows. Програма дозволяє легко зберігати важливу інформацію та відновлювати її у разі втрати.  

## Основні функції:  
1. Вибір папки для резервного копіювання  
2. Копіювання файлів у вказану папку  
3. Автоматичне створення резервних копій за розкладом  
4. Перевірка цілісності резервної копії  
5. Видалення старих копій для економії місця  
6. Відновлення файлів з резервної копії  
7. Журналізація операцій (логування)  
8. Мінімальний графічний інтерфейс  
 

Проєкт орієнтований на простоту та ефективність, що дозволить мінімізувати використання ресурсів та зробити програму легкою у використанні.
